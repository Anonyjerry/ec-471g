## 本配置文件有少许缺陷 请谨慎使用!! 建议不要使用本配置文件的dsdt

            ###'缺陷'
1.'原装的无线网卡ar9485无法驱动 国外也无解 建议购买其他网卡替换'

2.'摄像头无法识别 1.2M的像素超级渣'

3.'关机,重启有时候无法断电'
